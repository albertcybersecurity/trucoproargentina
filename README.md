# TrucoPro AR

Versión producción separada.

## Archivos incluidos

- `index.html`
- `styles.css`
- `app.js`
- `.nojekyll`

## GitHub Pages

Subí estos archivos a la raíz del repositorio y activá:

`Settings → Pages → Deploy from branch → main → /root`

## Cloudflare Pages

Framework preset: None / Static  
Build command: vacío  
Output directory: `/` o `.`

## Nota

Esta es una demo frontend completa. Para usuarios reales, pagos, partidas online reales y administración real, se necesita backend/base de datos.
